# Factory Metrics

This directory contains metric events generated by the factory at phase boundaries.

These JSON files are pushed to Plandek's Custom Time Series API by the CI/CD pipeline post-deploy step.

## Event Types

- `confidence-{jira-key}.json` — Phase 4 confidence self-assessment
- `outcome-{jira-key}-{task-group}.json` — Phase 5 execution outcome
- `swarm-{jira-key}.json` — Phase 6 hardening swarm completion
- `shipped-{jira-key}.json` — Phase 7 deployment event

## Pipeline

1. Factory writes JSON events here during execution
2. CI/CD post-deploy step reads all new JSON files
3. Pushes events to Plandek Custom Time Series API
4. Events appear in Plandek dashboards for overlay and correlation
