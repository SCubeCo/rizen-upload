# Factory Plans

This directory contains plans generated by the factory during Phases 1-4.

Plans are committed to git for:
- Audit trail
- Team learning
- Confidence accuracy tracking
- Plandek measurement

```

{jira-key}-phase2-slice-strategy.md
{jira-key}-phase3-{slice-name}.md
{jira-key}-phase4-{task-group}.md
```
